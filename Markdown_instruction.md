# Инструкция для работы с Markdown

## Выделение текста
Чтобы выделить текст курсивом необходимо обрмить его звёздочками (*).

Например *вот так*

Чтобы выделить текст полужирным необходимо обрамить его двумя звёздочками (**).

Например **полужирный**
